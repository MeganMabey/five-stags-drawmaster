# Five Stags Raffle Pro — Foundation

This is the clean foundation for the new Five Stags Raffle Pro application.

## Technology

- Phaser 4.2.1
- HTML/CSS user interface
- Responsive browser canvas
- GitHub Pages compatible
- No build tools or installation required

## Included in this foundation

- TV-friendly responsive game canvas
- Animated raffle drum
- Moving numbered balls
- Secure random winner selection
- No duplicate winners
- Full-screen mode
- Sound on/off
- Countdown
- Winner celebration
- Previous-winner history
- Browser saving
- Settings drawer
- Temporary vector-style mascot placeholder

## Next production steps

1. Replace the temporary mascot with the final Five Stags character artwork.
2. Improve drum metal, glass and lighting.
3. Add a physical winner-ball ejection path.
4. Add mechanical drum and crowd sound effects.
5. Add final particles, lighting and TV polish.

## Upload to GitHub

Replace your current main project files with:

- `index.html`
- `style.css`
- `game.js`
- `README.md`

The older files can remain in the repository, but this build does not use `script.js` or the old mascot PNG files.

After committing, wait for GitHub Pages to deploy and refresh with **Ctrl + F5**.
