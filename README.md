# Five Stags Drawmaster

A simple lucky-draw and raffle number generator for Five Stags.

## Features

- Set the highest ticket number each week
- Draws from 1 through to that number
- Prevents duplicate winners
- Saves the current draw in the browser
- Draw history
- Reset button
- Full-screen display
- Works on desktop, tablet and mobile
- No external libraries or internet connection required after loading

## Files

- `index.html` — page structure
- `style.css` — colours and layout
- `script.js` — number drawing and saved history

## Use it on your computer

Download all three files into the same folder, then double-click `index.html`.

## Put it on GitHub

1. Open your repository.
2. Click **Add file** → **Upload files**.
3. Upload `index.html`, `style.css`, and `script.js`.
4. Click **Commit changes**.

## Turn on GitHub Pages

1. Open the repository's **Settings**.
2. Choose **Pages** in the left menu.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch **main** and folder **/(root)**.
5. Click **Save**.

GitHub will then show the public website address.

## Add the real Five Stags logo later

Replace the temporary diamond mark in `index.html` with an image, for example:

```html
<img class="logo" src="five-stags-logo.png" alt="Five Stags">
```

Then upload the logo file into the same repository.
